

# ML Algorithm Visualiser

ML Visualiser is a web-based application designed to simplify the understanding of machine learning concepts through interactive visual representations. It enables users to explore how algorithms behave using dynamic graphs and real-time updates, making complex ideas more intuitive and accessible.

---

## Overview

The project focuses on improving conceptual clarity in machine learning by converting mathematical and algorithmic logic into visual formats. It is particularly useful for students and beginners who want to build strong foundational understanding through practical interaction.

---

## Features

* Interactive visualization of machine learning concepts
* Real-time updates for better understanding of model behavior
* Graph-based representation of data and outputs
* Clean and user-friendly interface
* Runs entirely in the browser without additional setup

## Getting Started

### Prerequisites

You only need a modern web browser such as Google Chrome, Edge, or Firefox.

---

### Installation and Setup

1. Clone the repository:

```
git clone https://github.com/Prateeks-0421/ML_VISUALISER.git
```

2. Navigate to the project directory:

```
cd ML_VISUALISER
```

3. Run the project:

* Open the `index.html` file in your browser
* Alternatively, use a live server (recommended for better performance)

---

### Running with Live Server (Recommended)

If you are using VS Code:

1. Install the "Live Server" extension
2. Right-click on `index.html`
3. Click on "Open with Live Server"

This will run the project on a local development server.
